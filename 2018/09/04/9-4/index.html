HTTP的简单介绍及查看请求与响应
2018-09-04|每周一篇

HTTP 请求包括哪些部分，如何用Chrome开发者工具查看 HTTP 请求内容

HTTP 的作用就是指导浏览器和服务器如何进行沟通。
一个请求一共分为4部分：

用 Chrome 发请求

  1. 打开 Network

  2. 地址栏输入网址

  3. 在 Network 点击，查看 request，点击「view source」

  4. 点击「view source」

  5. 如果有请求的第四部分，那么在 FormData 或 Payload 里面可以看到

HTTP 响应包括哪些部分，如何用Chrome开发者工具查看 HTTP 响应内容

一个响应一共分为4部分：

用 Chrome 查看响应内容

  1. 打开 Network

  2. 地址栏输入网址

  3. 选中第一个响应

  4. 查看 Response Headers，点击「view source」

  5. 会看到响应的前两部分

  6. 查看 Response 或者 Preview，你会看到响应的第 4 部分

如何使用curl命令

简介: curl是利用URL语法在命令行方式下工作的开源文件传输工具.

向百度发送一个post请求,请求包含数据 “1234567890”
curl -X POST -d "1234567890" -s -v -H -- "https://www.baidu.com"
其中-s表示silent模式，不显示进度和错误.
-v 展示curl的版本
-H 表示响应头
当然curl还有很多可以配置的命令，可以看见更多详细信息
