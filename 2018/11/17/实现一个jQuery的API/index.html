实现一个jQuery的API
2018-11-17|jQuery

细节记录：

因为是看完视频几天后做的题，就有点瞎写的感觉。
第一印象是用一个对象保存2个函数，就能直接调用了，这个思路是对的，但是写的时候错了。
对象应该用return返回，我却直接写成

还是代码写少了。
改正后还存在问题
我知道应该获取全部子节点用querySelectorAll,但是刚开始用console.log怎么都打印不出来。
然后最后google了一下，发现是自己忘记加document了。。。还是很尴尬的
然后偷懒用了forEach循环，再调用一个dom api就做好了添加类的功能，
同理替换文本也很简单。

收获：还是要多动手，不能用眼睛编程，很多东西看懂了但是自己打的时候很容易出错

要学的还有很多，急也急不来:)
