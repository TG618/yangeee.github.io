css3属性平滑加载以及消失【贝塞尔曲线】
2019-03-28|CSS

先上效果



实现

  最初看到这个是在学姐的博客上，屁颠屁颠跑去问效果怎么做的，结果一句模板就给打发了，好吧那就自己开搞。
  第一眼看过去就认为是动画做的，于是直接keyframe，然后一波操作确实拿下了效果，非常开心。可是当我移出鼠标的那一刻，炸裂！动画直接消失，根本就没有平滑效果，然后就是疯狂的补救，又是一顿操作，然而最后还是战败。不甘的我打开了Google，结果还是没有线索，但是知道了一个关键词：贝塞尔曲线
  最后没办法了，只能老老实实F12，令我震惊的是根本没有keyframe!找了半天跟动画有关系的只有一句transition-timing-function。脑袋突然顿悟，因为之前Google也看到了但是以为没用就忽略了，所以直接试，然后舒服的完成了上面的效果：）

  核心代码就三句，非常简单，果然是生命不息，折腾不止呀

transition-timing-function:ease-in-out;/* 贝塞尔曲线实现 */
transition-property:all;/* 规定什么属性发生动画 */
transition-duration: 2s;/* 动画时间 */
